# Schedule
上传 `Markdown` 格式的文件，一个星期按照以下格式 `commit` 一次，大家互相监督学习(。・∀・)ノ

### 格式

> 文件名：[filename].md --------------不做强制要求，能辨认即可

> 内容：
> [yyyy-MM-dd] Commit
>
>- 计划实践
>   - 这个星期是否有阅读专业相关书籍？
>     - 如阅读过，列举书籍、阅读量
>     - 知识吸收得如何
>     - 是否有相关的笔记，或者脑图，可填写 `Blog` 地址
>   - 这个星期是否有相关实践？
>     - 如有，简要列举
>   - 对下个星期的安排（可选）
>- 疑问（可选）
>    - [问题归属]
>      - 具体内容
